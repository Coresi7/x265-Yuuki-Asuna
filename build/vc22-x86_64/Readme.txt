Usage:
1st: install nasm and set NASM_ROOT point to the path of nasm executable.
2nd: clone lsmash and build it, then set LSMASH_ROOT point to lsmash dir.
3rd: install AviSynth, include FilterSDK
4th: install VapourSynth, include VapourSynth SDK

run cmd here, and run [path to vsdevcmd]
(could be found like this, "C:\Program Files\Microsoft Visual Studio\2022\Enterprise\Common7\Tools\VsDevCmd.bat")
and run multilib.bat
